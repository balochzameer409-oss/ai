# 3D Face Engine 🔥

## Deploy on Railway

### Step 1: GitHub پر upload کریں
1. GitHub پر نیا repository بنائیں
2. یہ سب files upload کریں:
   - app.py
   - requirements.txt
   - Dockerfile
   - railway.toml
   - templates/index.html

### Step 2: Railway پر deploy کریں
1. railway.app پر جائیں
2. "New Project" کلک کریں
3. "Deploy from GitHub repo" چنیں
4. اپنا repository چنیں
5. بس! Automatic deploy ہو جائے گا

### Step 3: Use کریں
- Railway آپ کو ایک URL دے گا
- اس URL پر جائیں
- تصویر upload کریں
- بٹن دبائیں اور 3D face دیکھیں! 🎉

## Features
- 468 face landmarks (MediaPipe)
- Real-time 3D rotation
- Left / Right / Up / Down
- Smile expression
- Wink effect
- Three.js WebGL rendering
